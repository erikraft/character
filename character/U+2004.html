Three-Per-Em Space

U+2004

Um espaço com largura de 1/3 de um em, usado em tipografia para criar espaçamento médio entre elementos. É mais largo que os espaços finos mas mais compacto que o espaço regular, ideal para separações moderadas em layout.

2. Design de Interfaces

Empregado em interfaces para espaçamento consistente entre botões e elementos.

3. Formatação de Listas

Utilizado em listas numeradas e bullets para criar espaçamento uniforme.

4. Layout de Tabelas

Aplicado em tabelas para separar colunas de forma consistente.
