ESPAÇO QUARTO

Four-Per-Em Space

U+2005

Um espaço com largura de 1/4 de um em, usado em tipografia para criar espaçamento moderado entre elementos. É mais fino que o espaço regular mas mais largo que os espaços especiais, ideal para separações sutis em layout.

Aplicações Práticas

1. Tipografia Editorial

Usado em jornais e revistas para criar separação sutil entre elementos de texto.

2. Design de Interfaces

Empregado em interfaces para espaçamento consistente entre elementos de formulário.

3. Formatação de Citações

Utilizado em citações e referências para criar espaçamento uniforme.

4. Layout de Documentos

Aplicado em documentos técnicos para separação consistente de seções.
